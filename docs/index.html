⚙️ SaneGit

AI-powered CLI for safer, smarter git workflows. Write better commits, catch risks, and work with confidence.

Features Commands Quick Start GitHub

✨ Key Features

📝 Smart Commits

Generate meaningful commit messages automatically using AI, with your approval before writing to git.

🔍 Change Explanation

Understand your changes in plain English with risk assessment and recommended next steps.

🛡️ Safe Pushes

Analyze your branch before pushing. Detect merge conflicts, CI failures, and safety concerns.

🎯 Repository Health

Run sg wtf to diagnose repo issues: conflicts, detached HEAD, failed CI, and more.

🔄 Undo & Fix

Revert commits safely or auto-fix common git issues with AI guidance.

⚡ Degraded Mode

All commands work offline and without AI. Use rule-based analysis when needed.

🎮 Available Commands

sg status

Show repository status, unstaged changes, and branch info.

sg explain

Explain staged changes in plain English with risk and next steps.

sg commit

Generate and create smart commit messages with confirmation.

sg push

Analyze branch before pushing. Warn about risks and conflicts.

sg check

Validate commit quality and adherence to best practices.

sg fix

Auto-fix common git issues with AI suggestions.

sg undo

Safely revert commits with planning and confirmation.

sg wtf

Diagnose repository health: conflicts, CI, HEAD state, and more.

sg ai-configure

Configure AI provider (OpenAI, Anthropic, Google, Mistral, custom).

🚀 Quick Start

1. Install

$ npm install -g sanegit npm install -g sanegit

2. Configure AI (Optional)

$ sg ai-configure $ sg ai-configure
Provider? (openai | anthropic | google | mistral | custom): openai
API Key or Environment Variable? OPENAI_API_KEY

3. Use SaneGit

$ git add src/feature.ts && sg explain $ git add src/feature.ts
$ sg explain
Summary: New authentication middleware
Risk: low
...

4. Commit Safely

$ sg commit $ sg commit
Proposed message: feat: add JWT auth middleware
Confirm? (y/n): y
[master abc1234] feat: add JWT auth middleware

⚡ Degraded Mode

Don't have AI configured? No problem! All commands work in degraded mode with rule-based analysis. Configure AI anytime with sg ai-configure to unlock AI features.

🎯 Why SaneGit?

Stop Manual Commit Messages

SaneGit analyzes your actual code changes and generates meaningful commit messages that follow best practices. No more generic "fix: stuff" commits.

Catch Risks Before They Happen

Know the impact of your changes before pushing. Get risk assessment and warnings about merge conflicts, CI failures, and unsafe patterns.

Works Everywhere

No AI? No problem. Use rule-based analysis offline. Add AI later with a single command.

$ git status On branch master
Changes staged for commit:
  modified: auth.ts (+45 lines)
  modified: config.ts (+12 lines)

$ sg explain Summary: Add JWT auth middleware
Risk: low
Recommendation: Include env var docs
Detail: Added secure token verification...
